# 七十二幻境诗意旅行 (72DreamlandPoetryTravel)

一款融合中国传统诗词文化与旅游探索的互动游戏，玩家通过猜诗词作者和填空来解锁全国各地的旅游胜地「72洞天」。

## 游戏概述

在这款游戏中，玩家将通过罗盘形式的界面选择一个洞天，然后需要完成诗词相关的挑战才能进入该景点。挑战包括识别诗词作者和填充缺失的字词。完成挑战后，玩家可以欣赏该景点的介绍、地图，并查看当地的天气预报。

## 核心功能

1. **幻境入口**：罗盘形式轮转展示72洞天和36诗仙
2. **诗词挑战**：
   - 识别诗词作者（从36位诗仙中选择）
   - 填充诗词中的缺失字词（随机隐藏3个字）
3. **洞天探索**：
   - 查看景点介绍和地图
   - 查看当地三天天气预报
   - 切换天气改变景点展示效果
   - 提供前往其他洞天的入口

## 数据来源

- 诗词数据基于开源项目 [chinese-poetry](https://github.com/chinese-poetry/chinese-poetry)
- 72洞天资料（国内热门旅游目的地）
- 36位著名诗人和词人资料

## 游戏玩法

1. 打开游戏，在首页罗盘界面选择一个随机洞天
2. 阅读展示的诗词，尝试识别作者并填充缺失的字词
3. 答对所有题目后，成功进入该洞天
4. 欣赏洞天风景，查看天气预报，可以选择前往其他洞天继续游戏
