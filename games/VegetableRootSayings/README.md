# 菜根潭摘录 - Vegetable Root Sayings

《菜根谭》是明代洪应明所著的经典修身养性箴言集，包含了丰富的处世哲学和人生智慧。这个Web应用提供了256句《菜根谭》中的经典名言，让您随时随地品味古人的智慧箴言。

## 功能特色

### 🎯 核心功能
- **随机浏览**：每次打开显示随机精选的名言
- **原文阅读**：点击名言卡片可查看完整原文段落
- **智能导航**：支持左右滑动、键盘导航等多种交互方式

### ❤️ 收藏功能
- **一键收藏**：点击心形按钮收藏喜欢的名言
- **收藏管理**：点击星形按钮查看所有收藏内容
- **本地存储**：收藏内容保存在浏览器本地，不丢失

### 📱 分享功能
- **智能分享**：支持Web Share API和剪贴板复制
- **社交媒体**：可分享到微信、微博等社交平台

## 使用指南

### 基本操作
1. **浏览名言**：页面加载后显示随机名言
2. **导航切换**：
   - 点击"上一句/下一句"按钮
   - 使用键盘左右方向键
   - 在手机上左右滑动屏幕
3. **查看原文**：点击名言卡片查看完整原文

### 收藏功能
1. **收藏名言**：点击心形按钮收藏当前名言
2. **查看收藏**：点击星形按钮查看收藏列表
3. **管理收藏**：在收藏列表中可移除不需要的收藏

### 分享功能
1. **分享当前名言**：点击分享按钮
2. **复制文本**：自动复制名言到剪贴板
3. **社交分享**：支持社交媒体直接分享

- `←` / `→`：上一句/下一句
- `空格` / `回车`：切换原文显示
- `Esc`：关闭原文显示
- `F` / `f`：收藏/取消收藏

## 部署说明

这个应用是纯前端应用，可以直接部署到任何静态网站托管服务：

### 本地运行
```bash

python -m http.server 8000

# 或使用Node.js服务器
npx http-server
```

## 数据来源

本应用包含的256句名言均出自《菜根谭》原著，每句都配有：
- **原文摘录**：精选的核心语句
- **完整出处**：详细的书名章节信息
- **原文段落**：完整的上下文内容
- **分类标签**：方便按主题查找
