# 64Hexagrams - 易经六十四卦记忆学习游戏

这是一个帮助用户学习和记忆易经六十四卦的互动游戏应用。通过游戏化的方式，让学习变得更加有趣和高效。

## 项目简介

该项目是一个纯前端的静态网站，使用HTML、CSS和JavaScript开发，无需后端支持。游戏通过匹配卦象的方式，帮助用户记忆易经六十四卦的名称、符号和基本含义。

## 功能特点

- 随机生成卦象配对游戏
- 显示当前进度和完成情况
- 提供卦象的详细信息查看
- 游戏成功后的结果展示
- 响应式设计，支持不同设备

## 安装与使用

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

## 开发

如果需要进行开发或修改：

```bash
# 进入项目目录
cd 64Hexagrams

# 安装依赖（可选，主要用于代码检查）
npm install

# 启动开发模式（提示如何运行）
npm run dev
```

## 项目结构

- `index.html` - 游戏主页面
- `styles.css` - 样式文件
- `script.js` - 游戏逻辑和卦象数据
- `.gitignore` - Git忽略文件配置
- `package.json` - 项目配置文件

## 游戏规则

1. 游戏板上会显示若干卦象卡片
2. 玩家需要点击卡片来选择相同的卦象进行配对
3. 每完成一组配对，进度条会更新
4. 当所有卦象都配对完成后，游戏结束并显示结果
5. 可以点击查看每个卦象的详细信息
