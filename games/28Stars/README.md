# 二十八星宿探秘

这是一款关于中国古代天文学中二十八星宿的教育类小游戏。玩家可以通过点击星象图中的星宿，查看其详细介绍，并将其归类到正确的方位（东方青龙、南方朱雀、西方白虎、北方玄武）。当集齐一方的所有七宿时，会有成功提示；当集齐所有四方共二十八星宿时，游戏完成。

## 游戏功能

- 以星象图形式展示二十八星宿
- 点击星宿可查看详细介绍
- 选择星宿所属的方位（东、南、西、北）
- 实时追踪各方星宿的收集进度
- 当一方星宿集齐时，显示成功提示
- 当四方星宿全部集齐时，庆祝游戏完成

## 技术实现

- 使用HTML5、CSS3和JavaScript开发
- 采用响应式设计，适配不同屏幕尺寸
- 使用SVG绘制星宿符号
- 动态生成星象图和背景星空效果
- 实现平滑的动画和过渡效果

## 开始游戏

1. 确保您的计算机已安装Python
2. 在项目目录下运行以下命令启动本地服务器：
   ```
   python3 -m http.server 8003
   ```
3. 打开浏览器，访问 http://localhost:8003

## 游戏玩法

1. 在星象图中点击任意星宿
2. 阅读该星宿的详细介绍
3. 选择您认为该星宿所属的方位（东方青龙、南方朱雀、西方白虎、北方玄武）
4. 如果选择正确，该星宿将被标记为已收集，对应方位的进度条会更新
5. 继续收集其他星宿，直到集齐所有二十八宿

## 二十八星宿简介

二十八星宿是中国古代天文学家为观测日、月、五星运行而划分的二十八个星区，是我国本土天文学的重要组成部分。它们分为四组，每组七宿，分别对应东、南、西、北四个方位，以及青龙、朱雀、白虎、玄武四种神兽。

- **东方青龙七宿**：角、亢、氐、房、心、尾、箕
- **南方朱雀七宿**：井、鬼、柳、星、张、翼、轸
- **西方白虎七宿**：奎、娄、胃、昴、毕、觜、参
- **北方玄武七宿**：斗、牛、女、虚、危、室、壁
