# 成语消除游戏

这是一个互动式成语学习游戏，玩家需要选择两个字块组合成四字成语，寓教于乐，帮助用户学习和记忆中国成语。

## 项目简介

该项目是一个纯前端的静态网站，使用HTML、CSS和JavaScript开发，无需后端支持。游戏通过字块选择和成语匹配的方式，让学习成语变得更加有趣和高效。

## 功能特点

- 随机生成字块供玩家选择组合
- 实时显示游戏进度和得分星级
- 提供游戏成功后的结果展示和重新开始功能
- 响应式设计，支持不同设备

## 安装与使用

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

## 开发

如果需要进行开发或修改：

```bash
# 进入项目目录
cd future

# 安装依赖（可选，主要用于代码检查）
npm install

# 启动开发模式（提示如何运行）
npm run dev
```

## 项目结构

- `index.html` - 游戏主页面
- `styles.css` - 样式文件
- `script.js` - 游戏逻辑和成语数据
- `.gitignore` - Git忽略文件配置
- `package.json` - 项目配置文件

## 游戏规则

1. 游戏板上会显示若干个字块
2. 玩家需要选择两个字块，尝试组合成四字成语
3. 每成功组合一个成语，进度条会更新
4. 根据完成时间和正确率，会获得相应的星级评分
5. 当所有成语都组合完成后，游戏结束并显示最终得分
