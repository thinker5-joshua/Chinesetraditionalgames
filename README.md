# 中国传统文化益智游戏群落

这是一个集合多个中国传统文化主题益智游戏的平台，旨在通过互动游戏的方式传承和弘扬中华优秀传统文化，让用户在轻松愉快的氛围中学习和体验传统智慧。

## 项目简介

中国传统文化益智游戏群落是一个纯前端的静态网站平台，目前包含两个特色游戏：

1. **易经六十四卦学习** - 通过配对游戏帮助用户记忆和了解易经六十四卦的相关知识
2. **成语消除游戏** - 通过选择字块组合成语的方式，提升用户的语言表达能力和成语积累

平台采用统一的设计风格，同时预留了未来扩展更多传统文化游戏的空间。

## 功能特点

- 统一的游戏入口和导航
- 响应式设计，支持不同设备访问
- 精美的视觉设计，融合传统文化元素
- 流畅的交互体验
- 易于扩展的架构设计

## 安装与使用

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件即可访问平台
3. 点击对应的游戏卡片进入相应的游戏

## 开发

如果需要进行开发或修改：

```bash
# 进入项目目录
cd Chinesetraditionalgames

# 安装依赖（可选，主要用于代码检查）
npm install

# 启动开发模式（提示如何运行）
npm run dev
```

## 项目结构

- `index.html` - 平台主页面
- `styles.css` - 样式文件
- `script.js` - 交互脚本
- `.gitignore` - Git忽略文件配置
- `package.json` - 项目配置文件

## 扩展指南

如果您想为平台添加新的游戏，请遵循以下步骤：

1. 在项目根目录下创建新的游戏文件夹
2. 开发游戏的HTML、CSS和JavaScript文件
3. 在主平台的`index.html`中添加新的游戏卡片
4. 更新README.md中的游戏列表
